# Getting Started

To run it locally

```bash
bun install
bun start
```

## Usual operations

Refer to the CI/CD job `client-testing` for the usual operations

# Techno stack

## React

We use React for the UI

## Shadcn

Components rely on [Shadcn](https://ui.shadcn.com).

## Routing

This project uses [TanStack Router](https://tanstack.com/router) as a file based router. Which means that the routes are managed as files in `src/routes`.
